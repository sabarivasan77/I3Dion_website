# Changelog

All notable changes to this project will be documented in this file.

## [v4.0.0] - Enterprise Single-Page Refine
- Added `CompanyIdentity` section detailing brand meaning.
- Refactored `Product` section to include a dynamic, animated expandable accordion (Framer Motion).
- Migrated Tech Stack and Project Roadmap into the Product accordion.
- Created `CompanyRoadmap` section for strategic milestones.
- Upgraded `Services` to display 8 industrial capabilities.
- Added comprehensive `README.md` and repository documentation.

## [v3.0.0] - Single Page Architecture Refactor
- Removed multi-page React Router logic for public pages.
- Consolidated all pages into a vertical scrolling single-page application.
- Added native smooth scrolling anchor links.

## [v2.0.0] - Enterprise DevSecOps
- Implemented MongoDB telemetry logging.
- Created `/admin` protected dashboard.
- Configured GitHub Actions for weekly Selenium testing.
- Hardened Vercel security headers.

## [v1.0.0] - Initial Migration
- Migrated vanilla HTML/JS site to React + Vite.
- Implemented Tailwind CSS and Three.js Hero background.
